Features

  • The host may now set a preference for whether reservers should auto roll on reserved items. Reservers may choose to always roll, never roll, or use the host preference.

Changes

  • Season of Discovery: Updated Scarlet Enclave loot.
  • Items in the Reserves tabs of the Reserve window are now sorted by drop location, and whether you have reserved the item.
  • Some tabs in the Reserves window now show the item drop location instead of the item classification underneath the item name.

Fixes

  • Fixed an issue where importing reserves wouldn't correctly set the number of reserves for players with higher/lower reserve limits.
  • Fixed items in Favorites tab of the Reserve window not properly sorting.
