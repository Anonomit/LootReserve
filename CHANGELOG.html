Features

  • Importing reserves no longer requires an item column to be present. This allows importing to be used just to set player-specific settings, like reserve limit and +1 score.
  • Hosts who intend to make LootReserve a required addon in their raid now have an option to disable to !reserve chat command. This prevents players from managing their reserves through chat.

Changes

  • Host window now always shows addon users, making it easier to check your own addon version when ungrouped.
  • Window sizes are now more aggressively restricted, and should scale up automatically if the minimum size has been increased beyond what a player has set.
  • Changed Mogu'shan Vaults nickname: Vaults -> MSV.

Fixes

  • Updated some fonts since Blizzard is removing them.
