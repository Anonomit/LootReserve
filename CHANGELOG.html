Features

  • Now supports +1. This features is disabled by default.
    • The implmentation assumes that any (non-reserved) 1-100 roll should be considered for +1.
    • The +1 feature can be toggled at any time, including during rolls.
    • Players' +1 values can be adjusted in the Players window.

Changes

  • Updated for Mists of Pandaria Classic. Raid loot data will be added later.
  • Season of Discovery: Crusader's Chalice is now treated the same as other items that players may want to win multiple times.
  • Wrath of the Lich King: Shadowmourne is now visible in the Quest Rewards section of ICC loot, and is not directly reservable.

Fixes

  • Fixed a potential lua error from some imported libraries that had not been updated recently.
